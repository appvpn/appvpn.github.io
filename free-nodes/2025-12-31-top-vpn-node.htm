  1. 点击右上角"+"号添加新配置

  3. 填写服务器信息：

  4. 高级设置中开启"混淆"选项（应对深度包检测）

性能优化技巧

要让台湾节点发挥最佳性能，还需要进行一些精细调整：

DNS优化

建议使用以下DNS组合：
- 主DNS：8.8.4.4（Google）
- 备DNS：1.1.1.1（Cloudflare）

这可以有效防止DNS污染，提升域名解析速度。

MTU调整

将MTU值设为1400-1450之间，可以减少数据包分片，特别适合跨境连接。

协议选择

不同网络环境下，各协议表现差异明显：
- 校园网/企业网：推荐V2Ray+WS+TLS
- 家庭宽带：SSR混淆效果更佳
- 移动网络：简单的SS协议反而最稳定

安全防护措施

使用代理工具时，安全问题不容忽视：

加密选择

避免使用已被证明不安全的加密方式（如RC4），优先选择：
- AES-256-GCM
- ChaCha20-Poly1305

流量混淆

启用协议混淆功能，使代理流量看起来像普通HTTPS流量，有效对抗DPI检测。

定期更换

即使是付费节点，也建议每3-6个月更换一次，防止长期使用被识别和封锁。

FAQ

Nekobox 可以与其他代理客户端共存吗？
可以。Nekobox 与 Shadowsocks、Clash 等客户端互不干扰，可以同时运行。用户可通过策略组或分流规则选择使用节点，实现灵活的网络管理。
Clash 如何实现智能分流？
Clash 通过结合规则文件和策略组，可以对不同网站或应用进行智能分流。例如将国内流量直连，海外流量通过代理节点，提高访问速度和稳定性，同时减少不必要的节点占用。
Potatso Lite 是否支持订阅更新？
Potatso Lite 支持通过订阅链接更新节点。用户在设置中输入 URL 后，应用会定期拉取最新配置文件，实现自动更新节点信息。这让日常使用更加便捷和稳定。
如何使用 Docker 部署 sing-box 或 v2ray？
选择官方或社区维护的 Docker 镜像，编写 docker-compose.yml 指定配置文件挂载、网络模式与端口映射。使用容器管理可以隔离运行环境、便于日志采集并与反向代理（如 Nginx）协同工作。
Mellow 的 TUN 模式与系统代理有何区别？
TUN 模式拦截系统 IP 层流量，实现全局代理，而系统代理通常只影响支持代理的应用。TUN 更适合游戏、视频和全局科学上网场景，保证所有流量通过节点，提高稳定性和速度。
Mellow 的 TUN 模式与系统全局代理有何区别？
TUN 模式直接拦截 IP 层流量，不依赖应用代理设置，系统全局代理通常只影响支持 HTTP/HTTPS 的程序。TUN 更适合全局科学上网和游戏环境。
Nekobox 如何分流多协议流量？
Nekobox 可以同时管理 Shadowsocks、VMess、Trojan 等多协议节点，通过策略组将不同流量分配到指定节点，实现精准分流，提高网络效率和管理便利性。
V2Ray 的 VMess over WebSocket 适合什么场景？
VMess over WebSocket 可以伪装成正常的 Web 流量，适用于被封锁较严格的网络环境。它通过将代理流量嵌入 WebSocket 协议中，使流量更难被检测和拦截。
VMess 配置 TLS+WebSocket 有哪些性能优势？
TLS+WebSocket 可伪装 HTTPS 流量，增强抗封锁能力，同时降低被封检测风险。结合策略组和节点延迟检测，可保证访问速度和连接稳定性，适合视频和游戏场景。
VLESS + Reality 模式如何隐藏流量特征？
Reality 模式通过伪装握手和真实网站特征，使流量难以被 DPI 检测。结合策略组使用，可有效隐藏节点通信特征，同时保持安全和高速传输，适合高封锁环境。
